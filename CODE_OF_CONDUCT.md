# Code of Conduct

Be practical, respectful, and specific.

- Discuss ideas and code, not personal attacks.
- Assume good intent, but be clear when something is unsafe or incorrect.
- Do not post private media, credentials, personal data, or exploit details in public issues.
- Maintainers may close comments or issues that are abusive, off-topic, or unsafe.

For sensitive reports, use the private vulnerability path in [SECURITY.md](SECURITY.md).
